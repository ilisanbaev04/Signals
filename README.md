#  Анализ одномерного сигнала
Программа для визуализации и анализа одномерных сигналов. Позволяет загружать сигнал из текстового файла или генерировать тестовый сигнал, автоматически находит и отображает на графике ключевые параметры.

##  Возможности
- **Средняя линия** – горизонтальная линия на уровне среднего арифметического.
- **Глобальный максимум и минимум** – наибольшее и наименьшее значение сигнала.
- **Локальные экстремумы** – все локальные максимумы и минимумы.
- **Точки перегиба** – точки смены знака второй разности.
- **Периодический участок** – поиск самого длинного повторяющегося фрагмента (с допуском).
- **Интерактивный график** – каждый элемент можно включать/отключать через чекбоксы.
- **Сохранение графика** – в форматах PNG, PDF, SVG.

##  Технологии
- Python 3.7+
- PyQt5 – графический интерфейс
- Matplotlib – построение графиков
- NumPy – численные расчёты

##  Requirments 
pip install -r requirements.txt
pip install PyQt5 matplotlib numpy
